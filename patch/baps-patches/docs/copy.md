# BAPS Copywriting Rules — The Watcher Edition

> P0-3 · 카피 톤 고침.
> 이 문서는 `docs/design-system.md` 의 "톤 앤 매너" 섹션을 **구체 예/반례** 로 확장합니다.
> LLM 으로 카피를 생성할 때 이 문서를 프리앰블로 주입하세요.

## 3대 원칙

| # | 원칙 | 의미 |
|---|------|------|
| 1 | **냉철함 (Cold)**     | 사용자를 달래지 않는다. 관찰한다. |
| 2 | **정교함 (Precise)**  | 숫자는 구체적으로, 문장은 짧게. |
| 3 | **위트 있는 독설**    | 웃기려 하지 않는다. 기록을 냉정히 해석한다. |

## 금지어

| ❌ 금지 | 이유 | 대체 예시 |
|---------|------|-----------|
| "완벽한" | Watcher 가 절대 쓰지 않는 낭만 형용사 | "정확한", "해독한" |
| "맛있게", "건강하게" | 결과를 미리 판단 | "기록 완료", "분석 완료" |
| "어서 오세요!", "화이팅!" | 감탄사 + 응원 | 중립 사실문 |
| "~했어요 🙂" | 이모지 + 감탄 + 반말 따뜻체 | "~했습니다." / "~됨." |
| "여유 있어요", "괜찮아요" | 위로조 | "정상 범위입니다.", "잔여 구간 여유." |

## 텍스트 구조 패턴

```
[사실 · 한 문장]. [제안 · 한 문장].
```

* 사실문은 **숫자를 먼저**, 뒤에 단위/비율.
* 제안문은 **권장 / 주의 / 중단** 셋 중 하나의 동사로 끝낸다.
* 이모지는 사용하지 않는다 (시간대 아이콘 🌅 등은 별도 SVG 교체 P2).

## 자동화 프리앰블 (LLM)

```
You are writing UI copy for BAPS — The Watcher Edition.
Tone: cold, precise, witty (never warm or encouraging).
Rules:
  - No exclamation marks. No emojis. No "~요" soft endings.
  - Prefer factual present ("~입니다", "~됩니다") or clipped ("~함", "~ 권장").
  - Start with a number when possible.
  - 2 sentences max. < 60 Korean characters per sentence.
Forbidden words: 완벽한, 맛있게, 건강하게, 화이팅, 힘내세요, 좋아요, 괜찮아요.
```
